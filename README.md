# CV Web – Daniel Hernán Peraza Yañez

Sitio web personal que presenta mi perfil profesional como desarrollador backend en formación, incluyendo experiencia laboral, formación académica, tecnologías y proyectos.

El objetivo del proyecto es contar con una versión pública, clara y profesional de mi currículum, desplegable como página estática y exportable a PDF.

## Descripción

Este sitio está construido sobre una plantilla base de HTML5 UP, personalizada y adaptada para reflejar mi experiencia en:

- Desarrollo backend con Spring Boot
- Diseño de arquitecturas MVC
- Modelado de bases de datos relacionales
- Administración básica de servidores Linux
- Proyectos creativos y experimentales

La estructura fue modificada para priorizar claridad, legibilidad y enfoque técnico.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- jQuery (incluido en la plantilla)
- Font Awesome (iconos)

## Secciones del sitio

- Perfil Profesional
- Experiencia Laboral
- Estudios y Certificaciones
- Tecnologías
- Infraestructura y Sistemas
- Logros Académicos
- Destrezas
- Intereses Personales y Profesionales
- Formulario de contacto

## Despliegue

El sitio puede ejecutarse simplemente abriendo el archivo `index.html` en un navegador o desplegarse en:

- GitHub Pages
- Servidor propio con Nginx
- Cualquier hosting estático

## Personalización

La información profesional se encuentra principalmente en:

- `index.html`
- `assets/css/main.css`

Las imágenes se ubican en la carpeta `images/`.
